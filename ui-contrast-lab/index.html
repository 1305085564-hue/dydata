排版 & 骨架

A 克制灰阶：排版即界面。用 gap/padding 替代 border，信息层级靠字号和字距建立。

B 白瓷光源：同样的排版骨架，但卡片带有内阴影折射，边缘更"透亮"。

C 暖米纸本：墨字落在牛皮纸上。文字偏暖褐（stone），像手作杂志的排版呼吸感。

D 纸本仪表 V2：Stone 全局迁移 + 7 档字号阶梯。正文 13px/22px，Eyebrow 10px 宽字距。

色彩 & 材质

A 克制灰阶：彩色只存在于 2px 导轨和状态点。按钮是实心暖橙，hover 靠位移体现。

B 白瓷光源：彩色是"光源"。按钮自带环境色漫反射，hover 时光晕扩散增强。

C 暖米纸本：暖橙是"印章"。印在暖米底上天然和谐，不需要光晕也能一眼识别。

D 纸本仪表 V2：彩色"镶嵌"不裸贴。状态点有 1px #FBFBF8 外圈描边（珐琅镶嵌），按钮有 1px #C96442 深边收边。

使用说明

  • 1. 点击顶部「全局底色」切换按钮，观察测试一与测试三在不同背景下的阅读舒适度差异。
  • 2. 将鼠标悬停在测试二的按钮上，感受方向 C 的「物理下沉」与方向 D 的「色彩呼吸」。
  • 3. 测试三的表格中，方向 C 和方向 D 的按钮成对出现，方便在密集信息环境下直接对比视觉噪音。
  • 4. 测试四方向 E：只有「发布日报」是暖橙色，其余所有交互仅靠灰阶 + 位移体现。滑动鼠标感受「板块在微微浮动」。
  • 5. 测试四方向 F：点击 Tab 和卡片切换选中态，观察幽灵线框（1px 暖橙边框 + 微弱阴影）与传统铺色高亮的质感差异。
  • 6. 测试三观察三种密度的「一屏信息量」与「视觉压迫感」；测试四点击图标切换 active 感受线框→实心的体积变化；测试五点击输入框感受焦点状态的差异；测试六直接对比三种浮层的「浮出方式」。
