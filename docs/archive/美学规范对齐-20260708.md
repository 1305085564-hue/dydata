# 美学规范对齐 · 改动总结

**日期**: 2026-07-08

## 目标
将 dydata 项目的色彩体系从 zinc 冷灰迁移到 stone 暖灰，字号收束到规范阶梯 (32/24/18/13/12)。

## 关键改动

### 1. CSS Token 层
- `design-tokens.css`: `--color-bg` #F0F0F1 → #FAFAF9 (stone-50)
- `tokens.css`: `--foreground` → #44403C (stone-700), `--background` → #FAFAF9
- 所有 hex 值对齐 stone 色板

### 2. 全局替换
- `zinc-` → `stone-` 批量替换 187 个文件
- 0 残留

### 3. 字号收束
- 10px/11px/9px/8px/14px/15px/17px → 规范三档 18/13/12
- 视频详情页仅使用 18px/13px/12px

### 4. 字深映射（按最新规范）
| Role | Before | After |
|------|--------|-------|
| 标题 | zinc-800 | stone-900 bold |
| 正文 | zinc-800 | stone-700 |
| 辅助 | zinc-500/400 | stone-500 |
| 元数据 | zinc-400 | stone-300 |
| 底色 | #F0F0F1 | #FAFAF9 |

### 5. UI 组件统一
Sheet/Dialog/Card 标题 → 18px bold stone-900
Table/Badge/Button/Input/Textarea 正文 → 13px stone-700
TableHead/SelectLabel → 12px stone-500
