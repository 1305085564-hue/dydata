# 美学标准

适用：阿禅名下所有项目（数据分析 / 话术库 / 团队中控 / AI 工作站）
哲学方向：`docs/美学准则.md`

本文只列量化规则与禁区。

---

## A 前置总则

- **基础层**（色彩 / 字号 / 圆角 / 间距 / 字重 / 边框）→ 正向定义，消灭随机性
- **组件层**（卡片 / 表格 / 按钮 / 徽章 / 进度条）→ 负面清单，底线之上允许克制惊喜
- 全文档中 × 开头的条目为不可破底线

## B 色彩系统

### B.1 底色三层

| 层级 | 色值 | 用途 |
|---|---|---|
| L0 | `#F4F4F5` zinc-100 | 页面背景 |
| L1 | `#FAFAFB` zinc-50 | 卡片、主面板 |
| L2 | `#FFFFFF` white | 焦点区、Modal、Focus 输入、激活 Tab |

层级只由底色差 + 1px `zinc-200` border 构成，× shadow 做层级

### B.2 文字灰阶

| 用途 | 色值 |
|---|---|
| 主文字 | `text-zinc-800`（× `zinc-950`） |
| 次要 | `text-zinc-500` |
| 元数据 / 空态 / 单位 | `text-zinc-400` |
| 分隔 / 边框 | `zinc-200`（优先用留白） |

### B.3 四语义色（统一光源，饱和 25-60 / 亮度 55-60）

| 语义 | 色值 | 场景 |
|---|---|---|
| 主 CTA | `#D97757` | 全站唯一主按钮 |
| 待处理 | `#D99E55` | 授权、生成中、提醒 |
| 成功 | `#6FAA7D` | 在线、通过、复制成功 |
| 异常 | `#C9604D` | 失败、驳回、违规 |

- CTA Hover 深色 → `#C96442`
- 动作 → 暖橙，状态 → 矿物色
- Primary 永远只有暖橙

**数据可视化色板**
```
#D97757 主橙   #E3B15C 琥珀   #7FA97E 绿
#8AA8C7 石青   #B08BB8 雾紫
```
- 折线：主橙 + 石青（最多 2 条主线）
- 柱状 / 分布：按顺序取色
- 网格线：`zinc-200` 虚线

### B.4 克制鲜艳（彩色形态铁律）

**核心铁律**：彩色只能以**点 / 线 / 按钮**三种形态出现，禁止任何面状彩色。每屏彩色总面积 ≤3%，且必须被留白包围。每屏 ≤1 个破格元素。

**允许的彩色形态**

| 形态 | 上色方式 | 举例 |
|---|---|---|
| 点 | ≤8px 实心圆，四语义色之一 | 状态点、列表项前置标识 |
| 线 | 1-2px 描边或左侧导轨，四语义色之一 | 2px 左导轨、outline 描边 |
| 按钮 | 主 CTA 实色 `#D97757` | 全站唯一主按钮 |
| 表单控件 | 复选框/单选框选中态 `#D97757` + 白勾 | ≤16px 小面积 |
| 数据可视化 | 主橙/琥珀/绿/石青/雾紫 | 仅图表内部 |

**禁止的面状彩色**

- × 彩色浅底 chip（`bg-emerald-50` / `bg-amber-50` / `bg-rose-50` / `bg-sky-50` 等任何 50/100/200 浅族底色）
- × 角色徽章 / 权限标签 / 状态 chip 的彩色背景
- × 彩色进度条已完成段（默认 `bg-zinc-300`，仅完成态可点睛 `#D97757` 或 `#6FAA7D`）
- × 彩色微图标底托

`emerald` / `amber` / `rose` / `sky` 四族仅限数据可视化图表内部，× 用于 UI 组件

**落地范例**
```tsx
// 正：outline + 左状态点
<span className="inline-flex items-center gap-2 rounded-lg border border-zinc-200 px-3 py-1 text-zinc-700">
  <span className="size-1.5 rounded-full bg-[#6FAA7D]" />
  已交
</span>

// 禁：彩色浅底 chip
<span className="bg-emerald-50 text-emerald-700 ...">已交</span>
```

## C 形态与节奏

### C.1 圆角（3 级 + Pill 特权）

| 层级 | 圆角 | Tailwind | 用途 |
|---|---|---|---|
| 外层容器 | 16px | `rounded-2xl` | 页面级主面板、大卡片（封顶 16px） |
| 内层卡片 | 12px | `rounded-xl` | 数据块、模块卡片、消息气泡 |
| 按钮 / 标签 / 输入框 | 8-10px | `rounded-lg` | 所有交互组件外壳（默认） |
| 状态点 / 头像 | 满圆 | `rounded-full` | 仅正圆元素 |

- × 非对称/异形圆角
- × 外层卡片超过 16px（`rounded-3xl` 禁用）
- 悬浮层（Dropdown / Modal）：`border border-zinc-200` + 柔光阴影

**Pill 特权（`rounded-full` 胶囊形）**

胶囊形是特权，× 通则：
- ✅ 登录/注册/落地页的核心 CTA
- ✅ 全站唯一终局锚点按钮
- × 列表项/表格/工具栏/次级操作按钮
- × 后台管理/数据分析页的日常按钮

判断口径：**一屏一声明**。已有 pill CTA 时，其他按钮必须 `rounded-lg` 让位。

### C.2 边框温度

| 温度 | 色值 | 用途 |
|---|---|---|
| 极冷 | `zinc-200` | 分隔线、卡片边界、表格横线 |
| 微温 | `zinc-300` | 可交互元素默认边框 |
| 暖橙 | `#D97757` | 激活/选中持久态 1px 轮廓 |

Focus Ring：`focus:ring-1 focus:ring-zinc-950/5`（× 暖橙做临时聚焦）

### C.3 字号 7 档

| 档位 | 字号 / 行高 | 字重 | 用途 |
|---|---|---|---|
| Display | `text-[32px] leading-[1.25]` | `font-semibold` | 核心数据大数、仪表盘顶格指标 |
| Section | `text-[24px] leading-[1.33]` | `font-semibold` | 模块标题、页面二级标题 |
| Title | `text-[18px] leading-[1.44]` | `font-medium` | 弹窗标题、小模块标题 |
| Body-L | `text-[14px] leading-[1.7]` | 默认 | 中文密集列表、长段正文 |
| Body | `text-[13px] leading-[1.7]` | 默认 | 默认正文（英文/数字密集） |
| Label | `text-[13px] leading-[1.23]` | `font-medium` | 表头、导航项、表单标签 |
| Caption | `text-[12px] leading-[1.33]` | 默认 | 脚注、辅助说明 |
| Eyebrow | `text-[10px] leading-[1.6]` | `font-medium` | 小眉题、分组标签（`uppercase tracking-[0.25em]`） |

- × `font-bold` / `font-black`，最高 `font-semibold`
- × `text-[16px]` 作为正文默认
- 数字/ID/图表数值全站强制 `tabular-nums font-mono`
- Display 档每屏最多 1 处，必须 ≥ 32px 留白包围

**数字与单位分层**
- 数值：`zinc-800 font-semibold text-[20px] tabular-nums`
- 单位（`%` `¥` `万`）：`zinc-400 font-medium text-[14px]`

### C.4 间距 6 档

| 档位 | 像素 | Tailwind | 用途 |
|---|---|---|---|
| XS | 4px | `gap-1` / `p-1` | icon 与文字、chip 内部 |
| S | 8px | `gap-2` / `p-2` | 相邻表单字段、小标签内部 |
| M | 16px | `gap-4` / `p-4` | 卡片内部、模块次级分组 |
| L | 24px | `gap-6` / `p-6` | 卡片 padding、主分组间 |
| XL | 32px | `gap-8` / `p-8` | 模块间、Section 之间 |
| 2XL | 40px | `gap-10` / `p-10` | 页面级主分区 |

优先用 6 档，紧凑交互元素（导航项、输入框、shadcn 组件内部）允许 `p-3` / `px-3`

### C.5 信息架构

**侧边栏**
- 浏览态：L0/L1 底色区分，建立方位感
- 深度工作态：提供隐藏/缩为图标栏的切换

**抽屉（临时侧边滑出）**
- 视觉区隔：`border border-zinc-200` + 底色差（L0 → L1）
- × 任何阴影

## D 交互与动效

### D.1 输入框（Soft Fill）

- 默认：`bg-zinc-100/70 border-transparent text-zinc-800`
- Focus：`bg-white border-zinc-200 shadow-sm focus:ring-1 focus:ring-zinc-950/5`

### D.2 动效曲线

```css
transition: transform 150ms cubic-bezier(0.4, 0, 0.2, 1),
            background-color 150ms cubic-bezier(0.4, 0, 0.2, 1),
            box-shadow 150ms cubic-bezier(0.4, 0, 0.2, 1);
```

- × `transition-all`
- × 弹簧过冲 overshoot

### D.3 Hover 静默

列表 / 次级按钮 / 导航 Tab 的 Hover：
- 仅底色变化：`hover:bg-zinc-100` 或 `hover:bg-white`
- × hover 位移（`-translate-y-[1px]`）
- × hover 阴影（`shadow-sm`）

### D.4 阴影使用规则（白名单制）

全站默认禁阴影。只有以下场景允许：

| 场景 | 阴影 |
|---|---|
| 主 CTA Hover | `0 2px 8px rgba(217,119,87,0.03)` |
| 居中独立卡（登录/Modal） | `0 12px 32px -16px rgba(15,23,42,0.08)` |
| 悬浮层（Dropdown/Tooltip） | `0 8px 24px -12px rgba(15,23,42,0.12)` |

不在白名单上的场景一律禁阴影。

## E 组件与模板

### E.1 空状态 / 加载

**空状态**：居中 `text-[13px] text-zinc-400` 文案 + 文案下方 ≤32px 微妙几何体（1px 细线框/小圆点阵列）。× 彩色插画 × spinner

**加载**：< 200ms 直出；≥ 200ms 骨架屏（`bg-zinc-100` 矩形）淡入替换

### E.2 列表 / 表格

- 斑马线：`bg-white` + `bg-zinc-50/50` 交替，或仅 Hover 当前行 `bg-zinc-100`
- 卡片底部元数据：`mt-6 pt-6` 留白分隔，× border-top

### E.3 高频组件

- **数据卡片**：`bg-white border-zinc-200 rounded-xl`，数值 `font-semibold tabular-nums`
- **Tabs**：默认 `text-zinc-500` 无底色；激活 `bg-white text-zinc-800 rounded-lg`（× shadow × 黑填充）
- **违规话术列表**：左侧 2px 实色 `#C9604D` 导轨，hover 整行 `bg-zinc-100`
- **消息气泡**：用户 `rounded-xl bg-zinc-950 text-white`；AI `bg-zinc-50 rounded-xl`
- **图标**：Lucide `stroke-[1.5]`，与文字 `gap-1.5`
- **分页器**：当前页 `border border-[#D97757]/40 text-[#D97757]`；非当前 `border-zinc-200 text-zinc-600`
- **角色徽章（outline + 左状态点）**：
  - 创始人：`border-[#D97757]/40 text-[#D97757]` + 点 `#D97757`
  - 管理员：`border-zinc-200 text-zinc-700` + 点 `#D99E55`
  - 组长：`border-zinc-200 text-zinc-700` + 点 `#6FAA7D`
  - 成员：`border-zinc-200 text-zinc-500`（无点）
- **进度条**：默认 `bg-zinc-300`；完成态 `bg-[#D97757]` / 达标 `bg-[#6FAA7D]`

### E.4 状态点四态

| 态 | 尺寸 | 填充 | 场景 |
|---|---|---|---|
| Critical | 8px | `bg-[#C9604D]` | 失败、驳回、严重告警 |
| Active | 8px | `bg-[#D97757]` | 当前步、进行中 |
| Done | 8px | `bg-[#6FAA7D]` | 完成、通过、在线 |
| Idle | 6px | `bg-zinc-200` | 未开始、无状态 |

Critical 唯一允许动效：单色呼吸 `opacity 0.6↔1, 2s ease-in-out infinite`
× `animate-pulse`

### E.5 左 2px 彩色导轨

标准配方：`border border-zinc-200 border-l-[2px] border-l-[#语义色]`

| 语义 | 色值 |
|---|---|
| Critical | `#C9604D` |
| Pending | `#D99E55` |
| Done | `#6FAA7D` |
| Primary | `#D97757` |

必须有导轨的场景：异常告警条 / 违规话术列表项 / 确认卡 / 分组区标题区

### E.6 黑色锚点

- 合法黑锚点统一 `bg-zinc-900`
- 每屏至多 1 个 `bg-zinc-900` 主按钮（终局动作：保存/提交/通过）
- 角色徽章 / 状态 chip / 分页器 × 黑底 → outline + 左状态点
- Tab 激活态：`bg-white text-zinc-800 rounded-lg`（× 黑填充）
