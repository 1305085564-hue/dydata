### 2. 同步 `perf.ts` 的类型修复（这是 loader 23 个报错的根因修复）
主工作区 `src/lib/perf.ts` 还是旧坏版。改动只有一处，把 `measureAsync` 的参数类型 `Promise<T>` 改成 `PromiseLike<T>`：
```ts
// 旧（坏）：
export async function measureAsync<T>(label: string, task: () => Promise<T>): Promise<T> {
// 新（对）：
export async function measureAsync<T>(label: string, task: () => PromiseLike<T>): Promise<T> {
```
根因说明：Supabase 查询构造器是 `PromiseLike`（thenable）不是完整 `Promise`，旧签名把它挡在门外，导致 loader 里所有查询结果类型退化成 `unknown`，级联出 23 个报错。改这一处，loader 无需改动即归零。
