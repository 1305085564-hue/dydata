## 四、设计原则速查（供 Claude 参考）

### [01] 基调去黑化与排版呼吸感

- **字体重量封顶**：`font-semibold` 为上限，配合 `tracking-tight`。禁止 `font-bold`、`font-extrabold`、`font-black`。
- **文字色值**：主文字 `text-zinc-800`，次要文字 `text-zinc-500`。彻底禁用 `text-zinc-950`、`text-black`。
- **全局底色**：`#F4F4F5` (zinc-100) 或 `#FAFAFB`。卡片底色用 `bg-zinc-50/50` 或 `bg-white` 仅在必要时使用。
- **排版替代边框**：用间距（gap/padding）和对齐区分信息，减少无意义 `border`。

### [02] 简约克制的鲜艳

- **色彩面积压缩**：绝不使用大面积彩色背景（如 `bg-red-50`、`bg-blue-500`）。
- **彩色唯一合法载体**：
  1. 状态点：2px–6px 实心圆点（`bg-emerald-500`）
  2. 极细线/边框：2px 提示导轨或 1px 选中边框
  3. 图标/文字强调：把 Icon 或数值本身变语义色
- **主高亮色唯一性**：整个页面只有主 Action 按钮使用 Claude 暖橙 `#D97757`。

### [03] 物理质感与表单组件

- **输入框默认态**：`bg-zinc-100/70 border-transparent`。Focus 时才变为 `bg-white border-zinc-200 shadow-sm`。
- **次级交互 Hover**：严禁使用彩色，只允许 `hover:bg-zinc-200/50` + `-translate-y-[1px]` + 微阴影。
- **悬浮层切割**：放弃重阴影，使用强硬 1px 边框 `border-zinc-200` + 极弱阴影 `shadow-sm`。

### [04] 默认回退参照

遇规范未覆盖的组件，参考 **Claude / Vercel / Linear**：先思考如何用纯文本和排版解决，其次考虑灰阶容器，最后才考虑加颜色。
